# Core Engine Files

This folder contains the UCS Engine source code.

| File | Description |
|------|-------------|
| `ucs_engine_lite.py` | Main UCS Engine — threat detection and routing |
| `protected_wrapper_lite.py` | Protected AI Wrapper layer (Layer 3) |
| `ucs_self_recognition.py` | Ethical self-regulation protocol (Layer 2) |
| `authentic_self_recognition.py` | Reasoning indicator module |
| `ucs_controlarena_integration.py` | ControlArena integration layer |
| `threat_patterns_sample.json` | Sample threat pattern library used in Layer 1 |

See the main [README](../README.md) for usage instructions.
